标题 说明/目标用户 位置

Dreamweaver 入门

Dreamweaver 概念和界面的基本介绍，附带详尽的初级教程。面向初级用户，以及希望了解新功能的中级和高级用户。

在线查看：http://livedocs.macromedia.com/go/livedocs_dreamweaver_cn

使用 Dreamweaver

在线查看：http://livedocs.macromedia.com/go/livedocs_dreamweaver_cn

获取 PDF：www.macromedia.com/go/dw_documentation_cn

扩展 Dreamweaver

阐述 Dreamweaver 框架和应用程序编程界面 (API)。适用于希望编译扩展功能或自定义 Dreamweaver 界面的高级用户。

在线查看：http://livedocs.macromedia.com/go/livedocs_dreamweaver_cn

使用 ColdFusion

收集了 ColdFusion 文档集中最重要的手册。（LiveDocs 上有完整的文档集。）适用于对 ColdFusion 感兴趣的所有人，包括初级用户和高级开发人员。

在 Dreamweaver 中查看：选择“帮助”>“使用 ColdFusion”

在线查看：livedocs.macromedia.com/go/livedocs_coldfusion/

获取 PDF：www.macromedia.com/go/cf_documentation

参考

HTML、服务器模型和其它类型的参考手册（主要由 O'Reilly 发行）。适用于需要进一步了解有关编码语法、概念等信息的所有人。

在 Dreamweaver 中查看：选择“帮助”>“参考”。要查看手册的完整列表，请单击“参考”面板中的“书籍”弹出菜单。
