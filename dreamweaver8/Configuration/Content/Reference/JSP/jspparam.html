<jsp:param>  

You can use the <jsp:param> action in the body of a <jsp:forward> or <jsp:include> action to specify additional request parameters for the target resource, as well as in the body of a <jsp:params> action to specify applet parameters.

Example:

<jsp:include page="navigation.jsp">
  <jsp:param name="bgColor" value="<%= currentBGColor %>"
    />
</jsp:include>
name String
Request-time value accepted: no

Mandatory. The parameter name.

value String
Request-time value accepted: yes

Mandatory. The parameter value.
