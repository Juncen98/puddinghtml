CEILING  
Description

Determines the closest integer that is greater than a specified number.

 
Returns

The closest integer that is greater than a given number.

 
Category

Mathematical functions

 
Function syntax
Ceiling(number)
 
See also

Int, Fix, Round

 
Parameters

 
Example
