INPUTBASEN  
Description

Converts string, using the base specified by radix, to an integer.

 
Returns

A number in the range 2-36, as a string.

 
Category

Mathematical functions

 
Function syntax
InputBaseN(string, radix)
 
See also

FormatBaseN

 
Parameters
