RTRIM  
Description

Removes spaces from the end of a string.

 
Returns

A copy of string, after removing trailing spaces.

 
Category

String functions

 
Function syntax
RTrim(string)
 
See also

LTrim, Trim

 
Parameters

 
Example

</cfif>
