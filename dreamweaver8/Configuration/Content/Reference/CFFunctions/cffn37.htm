BITXOR  
Description

Performs a bitwise logical XOR operation.

 
Returns

Bitwise XOR of two long integers.

 
Category

Mathematical functions

 
Function syntax
BitXor(number1, number2)
 
See also

BitAnd, BitNot, BitOr

 
Parameters

 
Usage

Bit functions operate on 32-bit signed integers, in the range -2147483648 - 2147483647.
