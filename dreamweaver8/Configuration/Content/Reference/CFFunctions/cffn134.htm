INT  
Description

Calculates the closest integer that is smaller than number. For example, it returns 3 for Int(3.3) and for Int(3.7); it returns -4 for Int(-3.3) and for Int(-3.7).

 
Returns

An integer, as a string.

 
Category

Mathematical functions

 
Function syntax
Int(number)
 
See also

Ceiling, Fix, Round
