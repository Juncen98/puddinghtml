508/W3C Accessibility Tests v. 2.0
Authors: UsableNet Inc.

Accessibility tests that implement the 508 Section of the U.S. Access Board and the priority 1 checkpoints specified by W3C/WAI-WCAG 1.0.
These tests are derived from those adopted in 508 Accessibility Suite vers. 1.2, a free Dreamweaver 4 extension, available from www.macromedia.com/exchange.
