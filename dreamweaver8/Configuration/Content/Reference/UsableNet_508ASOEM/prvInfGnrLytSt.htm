Provide information about site organization  
WAI / WCAG 1.0 Priority 2 checkpoint 13.3
 
 
Issue Description

If not present, add a site map to the site and describe how content is organized throughout the site. Highlight also the accessibility features that are available on the site.

 
 
How to check

Implement a site map either graphically, or as a set of nested lists (like in a table of content of a book), or use any other effective means for describing categories of information items.

If accessibility features implemented on the page are described, then users may configure the specific assistive technology that they are using to fit these solutions.

 
 
Issue Explanation

Consider a site map as the table of contents of the site. It is a powerful means for visitors to understand the layout of the site.

Consider also that for non-sighted users, experiencing a site takes much longer, since they have to listen to the content of the pages they visit. This takes much longer than just glancing it.
