Use last appropriate W3C technologies  
WAI / WCAG 1.0 Priority 2 checkpoint 11.1
 
 
Issue Description

This is a generic warning. It is best to always use the latest technologies released by the W3C that are appropriate for the task.

 
 
How to check

Review the latest W3C recommendations on the World Wide Web Consortium [http:w3.org] .

 
 
Issue Explanation

It is best to always code up to the latest standards, and let the browsers (and other technologies) catch up. In this way the developed pages do not need to follow the evolution of the browsers.

This means for example to use XHTML, CSS, SMIL (for multimedia captioning), MathML (for marking up mathematical formulas), XML and XSL (for completely separating content from presentation), P3P (for representing privacy information).
