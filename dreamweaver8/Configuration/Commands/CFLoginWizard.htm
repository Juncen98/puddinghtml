﻿ ColdFusion 登录向导

欢迎使用登录向导

此向导帮助您控制对现有应用程序的访问。要获得对应用程序的文件夹的访问，必须运行 ColdFusion 并正确配置活动站点。

请指定您要保护的目录的完整路径。该路径必须也在您的当前站点中。

验证类型

选择一个验证方法:

简单验证: 通过单一的用户名和密码保证应用程序的安全。

Windows NT 验证: 使用 Windows NT 用户名和密码保护您的应用程序。

LDAP 验证: 使用存储在 LDAP 服务器中的用户名和密码保护您的应用程序。

选择用户进行验证的方法:
使用 ColdFusion 登录页面。
使用弹出式对话框。
